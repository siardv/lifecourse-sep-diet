Baseline MM-Estimation Fits Before Outlier Adjustment

Author

Siard van den Bosch

Published

December 15, 2024

Modified

June 12, 2026

Purpose Produce baseline MM-estimation fits for Models 0–11 on the full sample
Inputs analysis_ready.sav
Outputs models.rds, unadjusted_data.rds, unadjusted_fit_{0..11}.rds, unadjusted_fits.rds
Method MM-estimation robust regression (bisquare ψ); bootstrapping disabled
Requires Step 6 (Post-Imputation Finalization)
NoteRationale
  • Energy adjustment: dhd_kcal enters as a covariate so that parental and partner SEP coefficients reflect dietary composition rather than caloric volume, consistent with the DHD15 scoring framework (Looman et al. 2017)
  • Lifestyle covariate: smoking is adjusted for as a co-occurring health behaviour; this follows standard practice in SES–diet research and is not attributed to a single source
  • Own SES at baseline: entering education_self and isei08_self in Model 1 yields parental and partner SEP coefficients interpreted net of own attainment, matching the wording of H1 and H2
  • Parallel composite and disaggregated families: Models 2–6 test mean-of-parents effects; Models 7–11 reproduce the same nesting with separate mother and father terms to expose parent-specific contributions

2.2 Model 1: Adult Covariates and Own SES (Adjustment Set)

Code 
# base model: demographics + lifestyle + own SES
base_formula <- dhd_index ~ woman + age + partner +
  education_self + isei08_self

# nutritional-epidemiology covariate: total energy intake
if ("dhd_kcal" %in% names(analysis_data)) {
  base_formula <- update(base_formula, . ~ . + dhd_kcal)
}
# lifestyle covariate: smoking status
if ("smoking" %in% names(analysis_data)) {
  base_formula <- update(base_formula, . ~ . + smoking)
}
# household economics (optional, conditional on availability)
if ("income_hh" %in% names(analysis_data)) {
  base_formula <- update(base_formula, . ~ . + income_hh)
}

model_1 <- base_formula

Code 
# collect model objects in numeric order
model_names <- ls(pattern = "^model_[0-9]+$")
model_names <- model_names[order(as.integer(sub("model_", "", model_names)))]
models <- lapply(model_names, function(nm) get(nm, envir = .GlobalEnv))
names(models) <- model_names

# save model specifications
saveRDS(models, file = file.path(models_dir, "models.rds"))

# prepare analysis data
unadjusted_data <- numeric_df(analysis_data) %>%
  dplyr::mutate(
    # ensure partner is numeric binary
    partner = as.numeric(partner == 1),

    # condition partner SES on having a partner (set to 0 for non-partnered)
    education_partner = dplyr::if_else(partner == 0, 0, education_partner),
    isei08_partner = dplyr::if_else(partner == 0, 0, isei08_partner)
  )
